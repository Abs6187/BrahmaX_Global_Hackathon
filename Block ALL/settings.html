Educational Content Filter

When enabled, the extension will analyze website content to determine if it's educational. Non-educational content will be blocked, while educational content will be displayed in a small frame.

The educational content filter uses Groq AI to analyze website content. You can get a Groq API key from console.groq.com.

About Pomodoro Technique

The Pomodoro Technique is a time management method developed by Francesco Cirillo in the late 1980s. It uses a timer to break work into intervals, traditionally 25 minutes in length, separated by short breaks.

1. Choose a task to work on

2. Set the timer for a work session (default: 25 minutes)

3. Work on the task until the timer rings

4. Take a short break (default: 5 minutes)

5. Repeat the cycle
